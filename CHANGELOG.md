# Changelog

All notable changes to Radish will be documented in this file.

## [0.1.1] - 2026-01-17

### Added
- Cloud telemetry integration (Supabase)
- Checkpoint data streams to central audit database
- Opt-out support via `RADISH_TELEMETRY=false` environment variable

### Changed
- Version bump to 0.1.1

## [0.1.0] - 2026-01-15

### Added
- Initial release
- Autonomous coding loop guardrails
- Violation detection (secrets, forbidden paths, commands, file limits)
- Git checkpoint integration
- Session logging and summaries
- Claude Code integration
